# סיכום צד השרת של הפרויקט

הפרויקט הוא שרת ASP.NET Core שמאפשר ניהול מערכת הגרלה/תרומות לבית ספר.

## עיקרון המרכזי של המערכת
הbackend בנוי כ-API מאורגן, מאובטח ומבוסס נתונים, שמחזיק את כל הלוגיקה העסקית של ההגרלה ומאפשר לאפליקציית Angular לתקשר איתו בצורה פשוטה ויציבה.

## מה עושה השרת
- מנהל משתמשים, תורמים, מתנות, כרטיסים, סלסלות וסטטיסטיקות.
- משתמש ב-EF Core עם SQL Server כדי לשמור ולקרוא נתונים.
- מספק autenticציה באמצעות JWT וחלוקת גישה לפי תפקידים.
- תומך ב-CORS, rate limiting, Swagger וקובצי לוג.

## משפטים מוכנים לראיון

1. זהו backend של מערכת הגרלה שמנהל נתונים, לוגיקה עסקית ואבטחה בצורה מאורגנת ומסודרת.
2. הפרויקט מדגים בנייה של API ASP.NET Core שמחבר בין ממשק המשתמש לבין מסד נתונים, עם JWT, שירותים וריפוזיטורי.
3. העיקרון המרכזי שלו הוא להפריד בין שכבות הלוגיקה, הנתונים והאבטחה כדי שהמערכת תהיה ניתנת להרחבה ותחזוקה.
4. אני רואה בזה מערכת שבה השרת אחראי על ניהול כל התהליכים של ההגרלה — משתמשים, מתנות, כרטיסים וסטטיסטיקות — בצורה מאובטחת ומסודרת.

## משפט קצר לשיחה בראיון
> זהו API של מערכת הגרלה שמנהל את כל הנתונים והלוגיקה העסקית בצורה מאובטחת, מאורגנת וניתנת להרחבה.
